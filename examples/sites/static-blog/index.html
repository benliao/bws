Virtual Hosting Architecture

Published: August 26, 2025

The BWS server now supports virtual hosting where multiple sites share the same port:

  • Multiple sites on the same port (8080)
  • Different hostnames for site routing
  • Separate static directories per site
  • Site-specific headers and configurations

Technical Details

Published: August 23, 2025
